### 5.3 TaskCreateModal（创建任务弹窗）

**功能描述**
- 以Modal弹窗形式配置并提交FIO测试任务
- 设备连接信息（IP、用户名、密码、设备路径）
- FIO测试参数（分基础+高级折叠展示）
- 选择故障注入类型
- null值参数不下发给后端
- 后端配置校验失败时弹窗展示错误详情

**基础表单字段**

| 字段组 | 字段名 | 类型 | 必填 | 默认值 | 说明 |
|--------|--------|------|------|--------|------|
| 设备配置 | device_ip | string | 是 | - | 设备IP地址 |
| | device_user | string | 是 | - | SSH用户名 |
| | device_password | password | 是 | - | SSH密码 |
| | device_path | string | 是 | - | 设备路径（如/dev/nvme0n1） |
| FIO基础 | rw | select | 是 | randread | 读写模式 |
| | bs | string | 否 | 4k | 块大小 |
| | size | string | 否 | 1G | 测试数据大小 |
| | numjobs | number | 否 | 1 | 并发任务数 |
| | iodepth | number | 否 | 32 | IO深度 |
| | runtime | number | 否 | 60 | 运行时间(秒) |
| | time_based | boolean | 否 | true | 基于时间完成 |
| 故障配置 | fault_type | select | 否 | none | 故障类型 |

**高级参数（折叠 Tab 展示）**

| 分组 | 参数 | 说明 |
|------|------|------|
| IO引擎 | ioengine | sync/libaio/io_uring等 |
| | direct | 直接IO |
| | sync | 同步IO |
| 缓冲设置 | fsync | 刷新间隔 |
| | buffer_pattern | 缓冲填充模式 |
| 随机IO | random_distribution | 随机分布类型 |
| | randseed | 随机种子 |
| 混合读写 | rwmixread | 读比例%(randrw) |
| | rwmixwrite | 写比例%(randrw) |
| 延迟控制 | thinktime | 思考时间(ms) |
| | latency_target | 延迟目标 |
| 限速设置 | rate | 带宽限制(KB/s) |
| | rate_iops | IOPS限制 |
| 数据校验 | verify | 校验算法 |
| | verify_fatal | 校验失败停止 |
| CPU/内存 | cpus_allowed | CPU亲和性 |
| | mem | 内存分配方式 |
| 统计设置 | stats_interval | 统计间隔(ms) |
| | log_avg_msec | 日志间隔 |

**组件结构**
```tsx
// components/TaskCreateModal/index.tsx
const TaskCreateModal: React.FC<{
  visible: boolean;
  onClose: () => void;
  onSuccess: () => void;
}> = ({ visible, onClose, onSuccess }) => {
  const [form] = Form.useForm();
  const { mutate: createTask, isLoading } = useCreateTask();
  const [advancedExpanded, setAdvancedExpanded] = useState(false);

  const handleSubmit = async (values: TaskForm) => {
    const fioConfig = buildFioConfig(values.config);
    try {
      await createTask({ ...values, config: fioConfig });
      message.success('任务创建成功');
      form.resetFields();
      onSuccess();
    } catch (error: any) {
      if (error?.response?.data?.error?.code === 'FIO_CONFIG_ERROR') {
        Modal.error({
          title: 'FIO配置错误',
          content: error.response.data.error.message,
        });
      }
    }
  };

  return (
    <Modal
      title="创建测试任务"
      open={visible}
      onCancel={onClose}
      footer={null}
      width={720}
      destroyOnClose
    >
      <Form form={form} layout="vertical" onFinish={handleSubmit}>
        <DeviceConfigSection />
        <FioBasicConfigSection />
        <Collapse
          activeKey={advancedExpanded ? ['1'] : []}
          onChange={() => setAdvancedExpanded(!advancedExpanded)}
        >
          <Panel header="高级配置" key="1">
            <FioAdvancedConfigSection />
          </Panel>
        </Collapse>
        <FaultConfigSection />
        <div style={{ textAlign: 'right', marginTop: 16 }}>
          <Space>
            <Button onClick={onClose}>取消</Button>
            <Button type="primary" htmlType="submit" loading={isLoading}>提交任务</Button>
          </Space>
        </div>
      </Form>
    </Modal>
  );
};
```

```typescript
// hooks/usePolling.ts
import { useEffect, useRef } from 'react';

interface PollingOptions {
  fn: () => void;
  interval: number;
  enabled: boolean;
}

export const usePolling = ({ fn, interval, enabled }: PollingOptions) => {
  const savedFn = useRef(fn);
  savedFn.current = fn;
  useEffect(() => {
    if (!enabled) return;
    const timer = setInterval(() => savedFn.current(), interval);
    return () => clearInterval(timer);
  }, [interval, enabled]);
};
```
